## ⚙️ Configuration

The first time you run RetroShelf, it scans your computer for game files. 

1. Open the File menu.
2. Select Add Library Folder.
3. Choose the folder where you store your game files.
4. Click Scan to let RetroShelf find your folders.
5. The application imports titles and adds cover art automatically.

## ❓ Frequently Asked Questions

**Does RetroShelf include game files?**
No. RetroShelf acts as a manager for files that you own. You must provide your own game files and emulators.

**Why does my game not launch?**
Check if you have the correct emulator installed for that system. Go to Settings and ensure the path to your emulator executable is correct.

**How do I update the app?**
RetroShelf checks for updates when you start the program. Follow the prompt if a new version exists.

**Can I use a controller?**
RetroShelf supports most standard controllers. Configure your controller inputs within your individual emulator settings. The application detects these settings automatically.
